##Swagger API

### 대기 순번 조회
대기 순번 조회는 Queue를 이용하여, waitingQueue(대기열) > ReadyQueue(입장) 로직으로 구현했습니다.
쿼리로 Position값을 안가지고와서 Swagger API에서 테스트 시 대기열 조회가 안됩니다.
